Command Line

hudl CLI

Full control over your Huddle01 Cloud resources from the terminal.

$ curl -fsSL https://get.huddle01.com/hudl | sh

Auto-detects OS and architecture. Installs to /usr/local/bin.

$ brew install huddle01/tap/hudl

Updates automatically with brew upgrade.

PS> irm https://get.huddle01.com/hudl.ps1 | iex

Run in PowerShell. Or download from GitHub Releases.

$ go install github.com/Huddle01/get-hudl/cmd/hudl@latest

Requires Go 1.25+. Binary installs to $GOPATH/bin.

AI Agents

MCP Server

67 tools for AI agents. Works with Claude, Cursor, Copilot, and more.

$ claude mcp add hudl -- npx -y @huddle01/mcp

Adds to Claude Code and Claude Desktop.

$ npx -y @huddle01/mcp

Run directly. No install needed.

$ npx -y @huddle01/mcp --install cursor

Auto-configures Cursor's MCP settings.

$ npm install -g @huddle01/mcp

Install globally. Then run hudl-mcp. View on npm.
